The Web

The web is programed or created by using HTML files that are linked by HTML.

HTML (Hyper Text Markup Language) is a basic for the web. The computer program that operates on the computer display HTML files that is found on the web, to link to the internet, then it connects to the servers where the host carries the files that make up the websites.

HTML

HTML can create the text content, markup, and references to other documents, such as images & videos, link to another web page.

HTML markup consists of an opening tag and closing tags. The entire piece of code is called an element. Attributes are listed after the elements, but before the corresponding value. Images and links are self-closing and are considered void tags, because they don't contain content.

More information about HTML

HTML Whitespace can turn everything into one line. Using br tag can seperate the content into multiple lines. Using p tag is another option but it is great for paragraphy.

Inline and Block elements are used to seperate content. An example of an inline element would be br tag because it just adds a break in the HTML line and wraps remaining text to the next line. An example of a block element would be p tag because it creates an invisible box around the content. Block elements also have a height and width.

Selectors

Selectors is where allows you to target specific HTML elements and apply style to them. This is helpful to use CSS to avoid unnecessary repetition that waste time. There are different types of Selectors:

  1. ID Selector
    Most powerful type of selector, beat out other types of selectors and styles. Having lower specificity selectors are easier to override.
  2. Class Selectors
    Most useful and versatile. Well supported in all browsers. Add multiple classes separated by space on HTML elements. Also JavaScript stuff you can do specifically for manipulating classes.
  3. Tag Selectors
    Most useful when changing properties that are unique to HTML element. Like setting the list style on a "ul" or tab-size on a "pre". Typically more useful to have class define styling that you can use on any HTML element.
  4. Attribute Selectors
    More useful than classes because they have the same specificity value, but they can be any attribute, not just "class". They also can have a value you can select by.
  5. Positional Selectors
    Several positional selectors beyond: nth-child; using simple expressions (like 3n="every third") you can select elements based on their position in the HTML.
  6. Pseudo Selectors
    ":empty" is one of many pseudo selectors, which you can recognize by the ":". They typically represent something you couldn't know just by the element and attributes alone. *These are different than pseudo elements, which are recognizable by "::"-they are responsible for adding things to the page by the things they select.
